# 安全报告 / Security

## 中文

请将可能导致 Vault 泄露、凭据暴露、越权文件访问、远程执行或数据损坏的问题发送至 `leochang210@gmail.com`，邮件标题使用 `Yunspire Security`。

报告应包含受影响版本、macOS 或 Windows 版本、最小复现步骤、影响说明，以及已脱敏的日志或截图。请不要附带真实密钥、私人知识内容或用户文件。

请先私下报告此类问题。未经授权，不要访问不属于你的数据，也不要进行破坏性、持久化或社交工程操作。

## English

Report vulnerabilities that may expose Vault data or credentials, permit unauthorized file access, enable remote execution, or damage data to `leochang210@gmail.com` with the subject `Yunspire Security`.

Include the affected version, macOS or Windows version, minimal reproduction, impact, and sanitized logs or screenshots. Do not include real secrets, private knowledge content, or user files.

Please report these issues privately first. Do not access data that is not yours or perform destructive, persistent, or social-engineering actions without authorization.
